# Computer Networks Lab
This repository contains Java programs implemented as part of the Computer Networks Lab.

## Contents
The repository includes implementations of various networking concepts and protocols such as:
- TCP Client and Server
- UDP Client and Server
- FTP Client and Server
- Stop and Wait Protocol
- Go-Back-N Protocol
- Selective Repeat
- Distance Vector Routing
- Leaky Bucket Algorithm
- Chat Applications
- Flow Control Mechanisms
- Congestion Control Algorithms

## Technologies Used
- Java
- Socket Programming
- TCP/IP Networking
